# 🚀 Enhanced Prompt Template for New AI Agents

**Simple, effective prompt that incorporates the new task system**

This template maintains the simplicity you want while leveraging the new dual-purpose task system for comprehensive assessment and system maintenance.

---

## 🎯 **Your Simple Prompt Template**

### **Basic Version (Recommended)**
```
🤖 AI Agent: Please read MAIN_GUIDE.md before conducting any work. After reading, you will be assigned a specific task from our maintenance system. The task will test your understanding while serving our current system needs.

🎯 ASSIGNED TASK: Brand Consistency Audit
📋 Task Details: Comprehensive brand compliance check of the entire codebase
 Source File: TASKS/BRAND_CONSISTENCY_AUDIT.md
📊 Expected Output: Detailed analysis with violations, fixes, and brand health score

🚀 Begin by reading MAIN_GUIDE.md, then execute the assigned task completely.
```

### **Even Simpler Version**
```
🤖 AI Agent: Please read MAIN_GUIDE.md before conducting any work. After reading, execute the task in CURRENT_TASK.md following the detailed instructions in TASKS/BRAND_CONSISTENCY_AUDIT.md.
```

---

## 🔄 **How to Use This Template**

### **Step 1: Check Current Task**
- **Open `CURRENT_TASK.md`** to see the currently assigned task
- **Note the task name** and source file location
- **Copy the task details** for your prompt

### **Step 2: Customize Your Prompt**
- **Replace task name** with current assignment
- **Update source file** reference
- **Adjust task details** if needed
- **Keep the format** consistent

### **Step 3: Deploy to AI Agent**
- **Copy-paste** your customized prompt
- **Agent reads MAIN_GUIDE.md** first
- **Agent executes assigned task** from TASKS directory
- **Agent provides comprehensive output** for assessment
